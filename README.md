# AWS CloudWatch Alerting Lab

Sample AWS monitoring project demonstrating CloudWatch alarms and SNS notifications for infrastructure alerting.

## Problem
Cloud environments need visibility into system health, performance, and failures to reduce downtime and improve incident response.

## Solution
This project demonstrates a basic monitoring setup using AWS CloudWatch alarms and SNS notifications to detect and respond to infrastructure issues.

## Services Used
- AWS CloudWatch
- Amazon SNS
- EC2 Monitoring
- JSON Configuration

## Skills Demonstrated
- AWS Monitoring
- Alerting & Incident Response
- Cloud Operations
- Reliability Engineering
- Infrastructure Visibility

## Example Use Case
Trigger alerts for high CPU usage, failed health checks, or abnormal system behavior.
